Shadows

Use shadows to highlight differences in elevation between two components.

Use one of these three shadow types:

Shadow X Y Blur Color Opacity CSS
10 0 1px 4px rgb(12, 12, 13); 10% box-shadow: 0px 1px 4px rgba(12, 12, 13, 0.1);
20 0 2px 8px rgb(12, 12, 13); 10% box-shadow: 0px 2px 8px rgba(12, 12, 13, 0.1);
30 0 4px 16px rgb(12, 12, 13); 10% box-shadow: 0px 4px 16px rgba(12, 12, 13, 0.1);
